# Infographic Creation

## Topic
Perfume Fragrance Pyramid

## Objective
To visually explain how a perfume fragrance evolves over time using an infographic design.

## Tools Used
- Canva

## Key Features
- Visual representation of the fragrance pyramid
- Clear categorization of Top, Middle, and Base Notes
- Use of icons and illustrations for better understanding
- Consistent color palette and typography
- Information design focused on clarity and readability

## Fragrance Pyramid Structure

### Top Notes
- Lemon
- Bergamot
- Orange

### Middle Notes
- Rose
- Jasmine
- Lavender

### Base Notes
- Vanilla
- Musk
- Sandalwood

## Design Principles Applied
- Visual Hierarchy
- Contrast
- Alignment
- Balance
- Information Storytelling

## Outcome
Created an infographic that simplifies the concept of perfume note progression through visual storytelling and structured information design.
